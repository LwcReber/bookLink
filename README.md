# bookLink
仿照简书部分功能的前后端、后台管理系统一体化的项目


web: 前端文件

server: 后端服务

admin: 后台管理系统

同时开启所有端运行命令

先在根目录`npm i`， 然后在根目录运行`npm run dev`。三个端开发环境就会同时运行
